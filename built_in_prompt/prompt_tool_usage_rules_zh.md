### 工具使用规则
1. 使用在线搜索工具时，`max_results` 参数每次查询最多不得超过 6 个结果。
2. 使用在线搜索工具时，保持 `query` 简短且基于关键词（2-6 个词为佳）。随着研究深度增加，查询应更加详细，即
研究越深入，查询应越详细。
3. 您可以操作的目录/文件系统是以下路径：{tmp_file_storage_dir}。切勿尝试在其他目录中保存/读取/修改文件。
4. 在进行在线搜索之前，优先使用本地资源。如果有 PDF 格式的文件，首先使用工具将其转换为 markdown 或文本格
式，然后以文本形式阅读。
5. 您基本上可以使用网络搜索工具搜索和检索您想了解的任何信息，包括财务数据、位置、新闻等。
6. 切勿使用 `read_text_file` 工具直接读取 PDF 文件。
7. 除非用户明确指定，否则不要以生成 PDF 文件为目标。
8. 不要使用图表生成工具来呈现旅行相关信息。
9. 如果工具生成了长内容，务必生成一个新的 markdown 文件来总结长内容并保存以备将来参考。
10. 当您使用 `write_text_file` 工具时，**必须始终**同时提供 `path` 和 `content` 参数。不要尝试一次性使用
`write_text_file` 写入超过 1 千个标记的长内容！！！
最后，在每次决定使用工具之前，仔细查看历史工具使用记录，避免因重复执行而浪费时间和 API 成本。请记住，您的
余额非常低，因此务必确保绝对效率。
